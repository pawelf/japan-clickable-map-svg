Japan Clickable Map SVG は、県毎にクリック可能な日本地図のテンプレートです。
日本地図にはインラインSVGを使用しているため、インラインSVGをサポートしているブラウザでお使いください。

Japan Clickable Map SVG is a template of a clickable map of Japan.
It's made using inline SVG.
Use the browser which supported inline SVG.
